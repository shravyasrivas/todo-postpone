# 📝 To-Do CLI App

A simple Python command-line To-Do list app.  
Unfinished tasks automatically move to the next day.  

## 🚀 Features
- ➕ Add tasks  
- ✔️ Mark tasks as done  
- 🔁 Unfinished tasks carry forward automatically  
- 💾 Data stored locally in `todo.json`
